# Software Architect Frontend Prompt

You are a senior frontend software architect.

Create a modern frontend boilerplate plan for a scalable web application using React, TypeScript, Tailwind CSS v4, Vite, and Vitest.

Write the response like an experienced architect preparing a clean implementation plan for an engineering team.

The plan should include:

## Project Stack
Explain the selected technologies and why they fit together.

Include:
- React 19
- TypeScript 5
- Vite 6
- Tailwind CSS v4
- Vitest 3
- React Testing Library
- jsdom

## Tailwind CSS v4 Setup Notes
Explain the important differences in Tailwind v4 setup.

Mention:
- CSS first configuration
- using @import "tailwindcss"
- using @tailwindcss/vite plugin
- avoiding old PostCSS based setup

## Suggested Project Structure
Provide a clean folder and file structure for the frontend project.

Include:
- src folder
- test setup
- vite config
- tsconfig files
- CSS entry
- example test file

## Key Configuration Files
Provide example configuration snippets for:
- vite.config.ts
- src/index.css
- src/setupTests.ts
- tsconfig.app.json

## Dependencies
Separate dependencies and devDependencies clearly.

Include all required packages for:
- React
- Tailwind v4
- Vitest
- Testing Library
- TypeScript
- Vite

## Additional Recommended Dependencies
Recommend useful frontend utilities for scalable UI development.

Include:
- react router dom
- clsx
- tailwind merge
- dnd kit packages

## NPM Scripts
Provide standard scripts for:
- development
- build
- preview
- test
- coverage

## Verification Steps
Provide a short checklist to confirm the boilerplate works correctly after installation.

The output should:
- be concise but practical
- use professional engineering language
- feel like a real frontend architecture document
- avoid unnecessary explanations
- avoid marketing style writing
- use markdown formatting
